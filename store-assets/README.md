# ストア用スクリーンショット生成

iPhone 8 の実機スクショ（750×1334）を素材に、App Store Connect の 6.9 インチ枠
（**1320×2868px**）のマーケ風スクリーンショットを生成する。実 UI のキャプチャを
埋め込むため審査規約 2.3.3（スクショは実アプリを反映）に適合する。

## 手順

1. 🧑 実機で以下を撮影し、AirDrop 等でこのリポジトリの `store-assets/raw/` に配置:

| ファイル名 | 撮る画面 |
|---|---|
| `01-onboarding.png` | 初回オンボーディング（再表示するには アプリ削除→dev build 再インストール か、シミュレータ） |
| `02-camera-ghost.png` | カメラ画面。ゴースト重畳 + シリーズチップ + 情報行が写る構図（被写体は植物などが映えます） |
| `03-series-picker.png` | シリーズ切替シート（2〜3シリーズ作った状態で） |
| `04-gallery.png` | ギャラリー（写真6枚以上ある状態） |
| `05-viewer.png` | 全画面ビューア（共有/削除ボタンが見える） |

2. AI: `template.html?slide=1..5` を Playwright（viewport 1320×2868）でレンダリング → `out/screenshot-1.png` 〜 `-5.png` を出力 → `sips` でサイズ検証
3. 🧑 目視確認 → App Store Connect の「6.9インチディスプレイ」枠にアップロード

コピー（キャプション）は `template.html` の `SLIDES` 配列で編集可能。
